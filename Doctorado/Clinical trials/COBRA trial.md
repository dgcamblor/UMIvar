# Description

The COBRA clinical trial, also known as NRG-GI005, is a phase II/III study that aims to evaluate the use of circulating tumor DNA (ctDNA) as a tool for guiding treatment decisions in patients with resected stage II colon cancer. The ctDNA analysis is performed using the LUNAR panel [@morrisNRGGI005COBRAPhase2020].
