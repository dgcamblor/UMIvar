Para estudiar la cobertura media de un archivo BAM:

```bash
samtools depth $bam | awk '{sum+=$3} END { print "Average = ",sum/NR}'
```

Para estudiar la cobertura entre múltiples archivos:

```bash
for bam in *.sorted.bam; do samtools depth $bam | awk '{sum+=$3} END { print "Average = ",sum/NR}'; done | awk '{sum+=$3} END { print "Average = ",sum/NR}'
```
