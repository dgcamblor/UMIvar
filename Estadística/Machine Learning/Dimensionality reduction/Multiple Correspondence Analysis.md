Performing dimensionality reduction on a dataset that contains a mix of categorical and numeric variables, is not attainable using traditional techniques like [[PCA (Principal Component Analysis)]]. For that purpose, this technique is useful.

## R

In **R**, there are several functions from different packages that allow us to apply Multiple Correspondence Analysis:

- **MCA** (FactoMineR)
- **mca** (MASS)
