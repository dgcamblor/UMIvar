El [[BIOS (Basic Input-Output System)]] o el [[EFI (Extensible Firmware Interface)]] carga el gestor de arranque (GRUB), que permite seleccionar el sistema operativo a cargar.

## Papel en *dual boot* con Windows

La idea principal es instalar Windows primero, y posteriormente utilizar Linux como un cargador de arranque de ambos.
