To ensure reproducibility, pulled GitHub pipelines are stored in a cache, that will subsequently be used even if the pipeline has been updated.

```bash
nextflow pull nf-core/methylseq
```

Some useful pipelines are:

- [methylseq](https://nf-co.re/methylseq)

## Arguments

nf-core arguments use double-hyphen (`--`), while Nextflow core arguments use a single hyphen (`-`)
