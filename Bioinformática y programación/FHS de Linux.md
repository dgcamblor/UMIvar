- /bin → Los binarios esenciales (e.g., cat, ls, cp). **No tocar.**
- /usr/bin → Reservado a paquetes instalados por managers de paquetes como `apt`. **No tocar.**
- /usr/local → Ejecutables compilados localmente. En el se colocan archivos únicos que permiten la ejecución de un programa.
- /opt → Programas con un sistema más complejo de carpetas. Por ejemplo, programas que se acaban de extraer de un tar. Para programas precompilados.

> Es bueno recordar que `/usr` no hace referencia a user, sino a unix system resources.
