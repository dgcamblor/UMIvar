### Zotfile

ZotFile es un gestor de archivos para zotero. ZotFile permite crear copias de los archivos y renombrarlos de acuerdo con sus metadatos.

Documentación disponible en: [ZotFile - Advanced PDF management for Zotero](http://zotfile.com/)

#### Enlazar con el dual-boot

> In Zotero (not Zotfile) preferences go to Advanced > Files and Folders. Make sure that Linked Attachment Base Directory is the same folder.

[Zotfile on multiple OS - Zotero Forums](https://forums.zotero.org/discussion/73148/zotfile-on-multiple-os)
#### Opciones ocultas

Zotero -> Preferencias -> Config Editor (/about:config)

**NOTA:** Cuando ZotFile copia un archivo, por defecto extrae el TOC (lo que cuenta como una nota adicional). Se puede desactivar o activar esta opción en `zotfile.pdfOutline.getToc`.
