Read groups are identifiers of a set of reads (generally, a group of reads generated under a single run of a sequencing instrument).

Some examples:

- `ID` = **Read group identifier**
- `PU` = **Platform Unit**
- `SM` = **Sample**
- `PL` = **Platform/technology used to produce the read**
- `LB` = **DNA preparation library identifier**

These labels are often required by downstream software, such as [[GATK]].
