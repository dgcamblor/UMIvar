**Finalidad.** Las órdenes límite sirven para entrar o salir de una posición a un determinado precio, mientras que las órdenes stop (stop-loss) son utilizadas para limitar pérdidas o asegurar ganancias.

**Ejecución.** 
- Las órdenes límite buscan comprar o vender a un precio determinado o mejor. Solo se pueden producir a dichos precios especificados.
- Las órdenes stop se activan cuando el precio de un producto ha llegado al especificado. Es entonces cuando esta orden se vuelve una [[Orden de Mercado]]. Al venderse a mercado, el precio de venta final puede no coincidir con el nivel marcado de stop-loss.
